# 事業計画書：AIエ-ジェントコマース『Synapse』

## 1. 事業概要 (Executive Summary)

**『Synapse』は、LLM（大規模言語モデル）の登場によって激変する消費者行動に対応する、次世代のAIエージェント・コマースプラットフォームです。**

従来のキーワード検索に依存したEコマース（Amazon, 楽天など）では、消費者は広告やブランドイメージに惑わされ、無限の選択肢の中から最適解を自力で探すという大きな負担を強いられてきました。

私たちは、この課題を根本から解決します。『Synapse』は、ユーザー一人ひとりに最適化されたパーソナルAIエージェントを提供。AIがユーザーの目的や価値観を深く理解し、広告やブランド知名度を完全に排除した**「客観的なデータと論理」**に基づいて、世界中の商品から「あなたにとって本当に価値のある選択肢」だけを提案します。

まず、MVP（実用最小限の製品）として**「プロテイン」**カテゴリに特化し、コアとなる仮説を検証。将来的にはあらゆる商品カテゴリを網羅し、AI時代の購買意思決定における、最も信頼されるインフラとなることを目指します。

## 2. 既存市場の課題と私たちのソリューション

### 2.1. 既存のEコマースが抱える根本的な課題

| 課題 | Amazonなど既存プラットフォームの問題点 |
| :--- | :--- |
| **① 広告・ブランド依存** | 検索結果は広告費や販売実績に大きく左右され、消費者は本当に優れた商品を客観的に判断できない。 |
| **② 意思決定の負担** | 無限の選択肢と不確かなレビュー情報により、消費者は「比較検討の地獄」に陥り、疲弊している（決定疲れ）。 |
| **③ "隠れた名品"との断絶** | 品質は高いが知名度の低い中小事業者の製品は、消費者の目に触れる機会が極めて少ない。 |

### 2.2. 私たちのソリューション『Synapse』

『Synapse』は、信頼のハブを「企業（ブランド）」から「パーソナルAIエージェント」へとシフトさせ、上記の課題を解決します。

| 解決策 | 『Synapse』が提供する価値 |
| :--- | :--- |
| **① 徹底した客観性** | AIエージェントは広告やブランド知名度を一切考慮せず、**客観的なスペックとコストパフォーマンス**のみで商品を評価・推薦します。 |
| **② 思考の外部化** | ユーザーは「考えながら話す」だけで、AIが思考を整理し、論理的な根拠と共に最適解を提示。意思決定のストレスから解放されます。 |
| **③ 発見（セレンディピティ）の創出** | AIの客観的な評価により、ユーザーはこれまで出会えなかった「隠れた名品」と出会い、中小事業者は公正な競争機会を得ます。 |

## 3. MVP（実用最小限の製品）計画

### 3.1. 検証したい中心的な仮説

> **消費者は、広告やブランドイメージを排除し、AIが提示する『客観的なデータと論理に基づいた推薦』を信頼し、購買を決定するようになるか？**

### 3.2. ターゲットカテゴリ：プロテイン

*   **選定理由:**
    1.  **ブランド依存が強い:** 大手ブランドが市場を席巻しており、ブランドスイッチが起これば仮説の強力な証明となる。
    2.  **スペックが明確:** タンパク質含有率や価格など、AIが論理的に比較しやすい。
    3.  **購買頻度が高い:** 検証サイクルを短期間で回せる。

### 3.3. MVPの仕様と開発手順

| ステップ | 期間 | アクション |
| :--- | :--- | :--- |
| **Week 1** | **開発** | **[DB構築]** Google Sheetsにプロテイン20製品のスペック、価格、Amazon URL、**ペルソナ・タグ**を手入力。<br>**[アプリ開発]** Python + Streamlit + Gemini API で、会話形式の推薦Webアプリを構築。 |
| **Week 2-3**| **検証** | **[ユーザー獲得]** X(旧Twitter)等でターゲットユーザー（筋トレ愛好家）を探し、DMで協力を依頼。最低10人にテストしてもらう。<br>**[データ収集]** アプリのログ（URLクリック等）と、1対1のオンラインインタビューで定性的なフィードバックを収集。 |
| **Week 4** | **分析** | 収集したデータに基づき、下記のKPIを評価し、次のアクション（Go/Iterate/Pivot）を決定する。 |

## 4. KPI（重要業績評価指標）と成功の定義

### 4.1. 測定する主要KPI

*   **無名ブランドURLクリック率: 30%以上**
    *   AIが有名ブランドと無名ブランドを並べて推薦した際、ユーザーが「無名ブランド」のリンクをクリックした割合。
*   **インタビューでの口頭コンバージョン率: 20%以上**
    *   インタビュー対象者のうち、「AIが推薦した無名ブランドを実際に買いたいと思ったか」という問いに「はい」と答えた割合。

### 4.2. 意思決定シナリオ

| シナリオ | 条件 | 次のアクション |
| :--- | :--- | :--- |
| **Go (進め)** | 上記KPIを両方とも達成。 | 仮説は検証されたと判断。データベースの拡張と、事業者との直接提携モデルの構築に着手する。 |
| **Iterate (改善)** | KPIのいずれか一方のみ達成。 | ユーザーインタビューを深掘りし、課題を特定。推薦ロジックやUIを改善し、再度検証サイクルを実行する。 |
| **Pivot (転換)** | KPIが目標を大幅に下回る。 | 仮説は棄却されたと判断。プロテイン市場の特殊要因か、モデル自体の問題かを見極め、方向転換を検討する。 |
